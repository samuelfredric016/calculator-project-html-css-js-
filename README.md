# 🧮 Calculator

A simple and responsive calculator built using **HTML**, **CSS**, and **JavaScript**. This project performs basic arithmetic operations with a clean and user-friendly interface.

## 📌 Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🟰 Calculate Result
- 🗑️ Clear Display (C)
- ⌫ Delete Last Character (DEL)
- 📱 Responsive Design
- 🎨 Modern UI with CSS

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 📂 Project Structure

```
Calculator/
│── index.html
│── style.css
│── script.js
│── README.md
```

## 🚀 How to Run

1. Download or clone this repository.

```bash
git clone https://github.com/your-username/calculator.git
```

2. Open the project folder.

3. Double-click **index.html** or open it in your web browser.

No installation or additional software is required.
